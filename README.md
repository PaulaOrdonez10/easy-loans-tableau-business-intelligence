# Easy Loans – Business Intelligence Dashboard (Tableau)

## Overview

This project was developed as part of the Master's in Data Science, Big Data & Business Analytics at Universidad Complutense de Madrid.

The objective was to design an interactive Business Intelligence solution using Tableau to analyse loan activity, merchant performance and business KPIs for Easy Loans during 2023.

The dashboard allows users to explore operational metrics, loan trends, merchant activity and geographical information through interactive visualisations and filters.

---

## Project Objectives

- Build a complete Business Intelligence solution using Tableau.
- Create executive KPI dashboards.
- Analyse loan operations across different countries.
- Evaluate merchant performance.
- Implement interactive filters and navigation between dashboards.
- Present business insights in a visual and user-friendly format.

---

## Key Insights

Some business insights identified during the analysis include:

- Significant differences in loan activity between countries.
- Concentration of loan volume among a small number of merchants.
- Clear seasonal patterns in loan creation.
- Merchant performance varies considerably depending on location and status.
